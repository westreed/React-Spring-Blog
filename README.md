# React.js + Spring 공부

공부하려고 만든 레포.<br>
이것저것 대충 끄적이는 용도입니다.

## 사용된 기술스택

* React.js
* React-Redux
* Spring
* SpringBoot
* JPA
* MySQL

## 사용된 오픈소스

* CKEditor5 (Post Editor)

## 서버 배포

```bash
serve -s build
java -jar -Dspring.profiles.active=prod 파일명.jar
```
